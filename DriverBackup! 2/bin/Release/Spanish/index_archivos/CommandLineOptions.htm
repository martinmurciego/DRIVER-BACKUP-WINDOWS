Es posible automatizar DriverBackup! utilizando unos parámetros en particular en la línea de comandos.

Los comandos y sus argumentos se pueden insertar en cualquier orden.

 

La sintaxis de los comandos serían así:

NOMBREDELCOMANDO=“ARGUMENTO”

 

Una lista de comandos disponibles:

 

Es obligatorio selecionar el modo para ello escribiremos:

 

MODE=“RESTORE”

Para selecionar el modo de Restauracion de dispositivos

 

MODE=“BACKUP”

Para selecionar el modo de Copia de seguridad o BackUp

 

LOG=“LOGFILENAME”

Opcional, eliga un archivo para la operación de registro. Si no se inserta ninguno se usará un nombre de archivo estándar.

 

OPT=“######”

Opcional, establece las opciones de los filtros en los modos Backup y Restauración. Cada opción permite unas características en particular. Si esta se omite se utilizará la configuración estándar OPT=“AL”

 

Opciones disponibles para todos los modos:

 

Dispositivo de filtrado:

A: Selecciona todos los dispositivos

M: Selecciona sólo los dispositivos OEM

H: Selecciona los dispositivos de otros proveedores

S: Selecciona sólo los dispositivos con firma digital

P: Selecciona los dispositivos plenamente compatibles con la copia de seguridad y restauración.

 

Registro

L: Habilitar el archivo de registro escrito.

 

Ejemplo:

OPT=“HSP” Configura DriverBackup! para seleccionar los dispositivos de otros proveedores con la firma digital y la plena compatibilidad. 

Modo BackUp o Copia de seguiridad

Los comandos disponibles para el modo de copia de seguridad son los siguientes:

 

BKPATH = "NOMBREDECARPETA"

Obligatorio , seleccionar ruta para la copia de seguridad.

 

BKFILE = "NOMBREARCHIVO"

Opcional, puede contener un nombre de archivo completo o una cadena con variable de formato.

 

BKPATHFMT y BKDEVFMT = "FORMATO"

Opcional, respectivamente ruta principal de la carpeta para la copia de seguridad y ruta de la carpeta para la copia de seguridad de un único dispositivo.

 

BKDESC=“DESCRIPCIÓN”

Opcional, establece una descripción para la copia de seguridad.

 

OPT=“######”

La opciones ampliadas para el modo de copia de seguridad disponibles con el comando "OPT" son los siguientes:

 

R: Generar los archivos necesarios para la restauración automática de los controladores.

W: Permite la sobreescritura de los archivos.

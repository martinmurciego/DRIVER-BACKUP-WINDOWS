Instalación y compatibilidad

DriverBackup! 2.0 es compatible con Windows XP o posteriores.

Si usted lo usa en Windows Vista, simplemente se requiere una "copiar y pegar". Si usted lo usa en Windows XP o similar debe estar instalado .NET Framework 2.0

Si no desea instalar el .NET Framework puede utilizar una versión anterior de DriverBackup!

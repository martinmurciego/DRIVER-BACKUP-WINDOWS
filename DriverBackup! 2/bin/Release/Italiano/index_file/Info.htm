Installazione & compatibilità

DriverBackup! 2 è compatibile solamente con Windows XP o versioni superiori.

Per il funzionamento su Windows Vista non è richiesto alcun pacchetto aggiuntivo, ma solamente il semplice “Copia e Incolla”. In Windows XP e altre versioni diverse da Windows Vista è necessario scaricare ed installare il .NET Framework 2.0, scaricabile dal sito Microsoft.

 

Anche se l’esecuzione su sistemi precedenti a Windows Vista può risultare scomoda ho preferito uniformare il programma ai sistemi moderni, vantaggio al momento poco apprezzabile, ma che sicuramente sarà importante in futuro.

 

Per chi volesse operare su sistemi obsoleti è ancora disponibile DriverBackup! 1.0.3, sicuramente molto semplice, ma efficace.

DriverBackup! Permette all’utente di controllare la formattazione dei percorsi di backup usando dei comodi segnaposto.

 

Per qualsiasi directory (Percorso principale di backup,file bki,percorso dispositivo)

 

%COMPUTERNAME%  Sostituito con il nome del computer

 

%NOW%  Data e ora corrente, formattata secondo le impostazioni indicate dall’utente.

 

Solo per il singolo dispositivo

 

%DEVNAME% Nome del dispositivo

%CLASSNAME% Nome della classe di provenienza

%CLASSGUID% Identificatore di classe (GUID)

%PROVIDERNAME% Provider name

 

Home

Qualche esempio:

 

Percorso principale: “Drivers %COMPUTERNAME%” —> “Drivers PC-OBELIX”

 

Percorso del dispositivo: “%DEVNAME% of %PROVIDERNAME%” —> “Graphic adapter 9330S of Best Graphics”.

La formattazione della data e dell’ora corrente può essere controllata utilizzando i normali specificatori di formato di Windows.

Qualche esempio:  "09/27/2006 15:06:01"

D  mostra "09/27/2006"
D  -> "Martedì, 27 Settembre 2006" 
G  -> "09/27/2006 14:15:39"

MM/dd/yyyy ->  09/27/2006

MMMM,dd yyyy -> Settembre ,27 2006

Nella schermata principale del programma compaiono tutti i dispositivi installati sul computer locale, ordinati per classe di appartenenza.

Normalmente vengono visualizzati tutti i dispositivi, ma è possibile filtrare la lista impostando una classe di produttori specifica o agendo sui pulsanti “Piena portabilità” e “Firma digitale”.

 

Per selezionare una classe di produttori è sufficiente fare clic su uno dei tre pulsanti posti al di sopra della lista.

 

-Tutti:  Visualizza tutte le periferiche

-OEM: Visualizza le periferiche con drivers prodotti da Microsoft e incluse in Windows

-Terze Parti: Visualizza le periferiche con drivers prodotti da terzi.

 

I due pulsanti a destra filtrano rispettivamente le periferiche pienamente compatibili al Backup e al Ripristino, che non presentano problemi (es: files non trovati), e le periferiche

dodate di firma digitale WHQL, certificate da Microsoft.

 

 

A questo punto e sufficiente selezionare i dispositivi a cui si è interessati e successivamente premere il pulsante “Backup”.

 

Nella finestra che compare selezionare una directory in cui verranno copiati i files tramite il pulsante “Sfoglia”. Si consiglia di scegliere una

directory vuota.

 

E’ possibile inoltre specificare il formato con cui verranno formattati il percorso principale

e le directory dei vari dispositivi o inserire una descrizione.

I specificatori di formato disponibili sono descritti qui.

 

Alla fine delle operazioni verrà generato un file di informazioni “.BKI”, indispensabile per il successivo ripristino.

 

Le due caselle in basso permettono rispettivamente di abilitare la sovrascrittura di files

eventualmente presenti nella directory di backup e di creare i files necessari al ripristino automatico dei drivers.

 

I files per l’auto-ripristino comprendono un file Batch eseguibile da riga di comando e

un files “autorun.inf” che permette di avviare automaticamente il ripristino da CD\DVD-ROM o Flash USB.

 

Affiché “autorun.inf” faccia il suo compitò è necessario però copiare i files eseguibili,la cartella in cui è stato effettuato il backup e il file “.bki” contenente le informazioni sul backup nella cartella principale del dispositivo di memorizzazione.

 

 

 

 

 

 

Nota:

 

Alcuni dispositivi potrebbero essere contrassegnati come incompatibili o compatibili solo parzialmente al Backup e al Ripristino. Questo è dovuto al fatto che DriverBackup! non è

riuscito a ricollegare tutti i files necessari o la configurazione del dispositivo suggerisca

che potrebbe essere adatto solo al sistema operativo in cui è installato.

E’ possibile automatizzare DriverBackup! tramite le sue opzioni da riga di comando.

I comandi con i loro argomenti possono essere inseriti in modo casuale.

 

La sintassi standard di un comando é  NOMECOMANDO=“ARGOMENTO”

 

Sotto sono riportati tutti i comandi disponibili:

 

MODE=“RESTORE” o MODE=“BACKUP”

Obbligatorio, seleziona la modalità ripristino o backup

 

LOG=“LOGFILENAME”

Opzionale, specifica il nome del file in cui verrà effettuato il logging delle operazioni. Se non è presente il programma sceglie un nome di default.

 

OPT=“######”

 

Opzionale, specifica le opzioni di configurazione secondarie. Ciascuna lettera abilità una particolare opzione. Se il comando non è presente verrà adottata la configurazione predefinita OPT=“AL”

 

Opzioni disponibili per tutte le modalità di funzionamento:

 

Filtraggio dei dispositivi:

A: Seleziona tutti i dispositivi

M: Seleziona i dispositivi con driver Microsoft

H: Seleziona i dispositivi con driver proprietari.

D: Seleziona i dispositivi automaticamente. (Solo modo Ripristino)

S: Seleziona i dispositivi con firma digitale.

P: Seleziona i dispositivi con piena compatibilità al ripristino e al backup

 

Logging

L: Abilita il logging su file.

V: Abilita la modalità silenziosa. Non verrà chiesta alcuna conferma all’utente.

 

Esempio: OPT=“HSP” Seleziona i drivers con produttori diversi da Microsoft, firma digitale e massima compatibilità al ripristino.

 

 

 

 

 

 

Modalità ripristino

I comandi per la modalità ripristino sono:

 

BKFILE=“NOMEFILE”

 

Richiesto, specifica il nome del file di backup (INF). Se non è specificato un percorso il programma cercherà nella sua directory principale.

 

Le opzioni disponibili tramite il comando “OPT” sono:

 

U: Aggiorna i file del driver senza chiedere conferma all’utente.

N: Aggiorna la configurazione hardware del computer dopo il ripristino.

Modalità Backup

Ecco i comandi disponibili per la modalità backup:

 

BKPATH=“NOMEPERCORSO”

 

Necessario, specifica la cartella principale di backup.

 

BKFILE=“NOMEFILE”

 

Opzionale, può contenere un nome file completo o una stringa con specificatori di formato.

 

BKPATHFMT e BKDEVFMT =“FORMATO”

 

Opzionale, impostano rispettivamente il formato con cui verranno generate la cartella principale di backup e la cartella del singolo dispositivo.

 

SYSPATH=“PERCORSOSISTEMA”

 

Opzionale, usato solo nel backup offline. Specifica il percorso del sistema offline.

 

BKDATEFMT=“mm\dd\yy”

 

Opzionale, specifica la formattazione della data.

 

BKDESC=“DESCRIZIONE”

 

Opzionale, imposta una descrizione per il backup.

 

Options for backup mode available with “OPT” command are:

 

R: Crea i file per il ripristino automatico.

W: Abilità la sovrascrittura dei files presenti sul percorso di backup.

O: Solo per il backup offline, sostituisce %COMPUTERNAME% con il nome del computer offline.

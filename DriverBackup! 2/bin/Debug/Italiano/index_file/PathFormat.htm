DriverBackup! permette di formattare i nomi delle cartelle in cui verranno salvati i files dei vari dispositivi di cui effettuare il Backup.

 

Sono disponibili vari segnaposti che verranno sostituiti nella formattazione

del percorso.

 

Utilizzabili ovunque (Directory principale di backup, file informazioni,singolo dispositivo)

 

%COMPUTERNAME%  Nome del computer

 

%DATE%  Data corrente

 

Solo formattazione del percorso del singolo dispositivo

 

%DEVNAME% Nome del dispositivo corrente

%CLASSNAME% Classe di appartenenza del dispositivo

%CLASSGUID% Identificatore della classe di appartenenza

%PROVIDERNAME% Nome del produttore

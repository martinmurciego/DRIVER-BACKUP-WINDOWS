E’ possibile automatizzare DriverBackup! all’interno di script o semplicemente richiamarlo da riga di comando.

 

I comandi possono essere inseriti in ordine causuale, ma in base alla

modalità, alcuni devono essere obbligatoriamente inseriti.

 

La sintassi di un comando è  NOMECOMANDO=“ARGOMENTO”

 

Ecco una lista dei comandi disponibili:

 

MODE=“RESTORE” o MODE=“BACKUP”

Obbligatorio, seleziona la modalità di funzionamento.

 

LOG=“NOMEFILELOG”

Opzionale, seleziona un file in cui verrà effettualo il logging delle operazioni. Se non è presente verrà utilizzato il nome di default.

 

OPT=“######”

 

Opzionale, imposta le opzioni secondarie al backup o al ripristino.

Ogni singola lettera della stringa argomento attiva una particolare opzione.

Se il comando non è presente verrà utilizzata la configurazione di default

OPT=“AL”.

 

Le opzioni disponibili per qualsiasi modalità sono:

 

Filtraggio periferiche:

A: Tutti i dispositivi

M: Oem

H: Terze Parti

S: Dispositivi dotati di firma digitale

P: Dispositivi dotati di piena compatibilità al Backup e al Ripristino

 

Logging

L: Attiva il logging delle operazioni effettuate

 

ES: OPT=“HSP” Configura DriverBackup! per il salvataggio di tutte le periferiche con drivers prodotti da terzi, dotati di firma digitale e piena compatibilità.

 

 

 

 

 

 

Modalità Ripristino

I comandi utilizzabili per la modalità Ripristino sono:

 

BKFILE=“NOMEFILE”

 

Obbligatorio. Seleziona il file di backup da aprire. Se il nome non contiene un percorso DriverBackup! cercherà il file nella propria directory.

 

RSPATH=“PERCORSO”

 

Opzionale. Seleziona la directory in cui verranno copiati i files. Se la cartella non esiste

verrà creata al momento del ripristino. Nel caso in cui il comando non sia presente

verrà utilizzata la directory di default “%windir%\DriverBackupRestore”.

 

Le opzioni disponibili per la sola modalità Restore sono:

 

U: Aggiorna la configurazione del driver quando già installato.

N: Cerca nuove periferiche Plug n Play dopo il ripristino.
